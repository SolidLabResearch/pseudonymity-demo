## Testing

```bash
npm install
```

### Components

Run the tests as follows:

```bash
npm run test:components
```

To teardown the test infrastructure, execute:

```bash
npm run test:components:teardown
```

### Use cases

Run the tests as follows:

```bash
npm run test:usecases
```

To teardown the test infrastructure, execute:

```bash
npm run test:usecases:teardown
```
